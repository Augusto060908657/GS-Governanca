# Global Solution 2026.1 - FIAP | Inteligência Artificial

Este repositório tem como objetivo centralizar todas as entregas desenvolvidos para a **Global Solution (GS) de 2026.1** do curso de Inteligência Artificial da FIAP. 

## Sobre o Projeto e Integração

A proposta desta entrega é demonstrar a integração e o funcionamento de todos os módulos técnicos desenvolvidos ao longo da Global Solution 

Cada diretório deste repositório representa uma disciplina específica. O objetivo final é mostrar como essas diferentes frentes de tecnologia (desde computação quântica até robótica e visão computacional) se conectam em uma solução coesa.

## 🗂️ Estrutura do Repositório

Os trabalhos foram divididos de acordo com as disciplinas do semestre. Navegue pelas pastas abaixo para acessar os códigos e arquivos de cada módulo:

* 📁 `AI for Robotic Process Automation`
* 📁 `Cluster Computing, Computação Neuromórfica...`
* 📁 `Computação Quântica & IA`
* 📁 `FRONT END & MOBILE DEVELOPMENT`
* 📁 `Generative AI & Advanced Nets`
* 📁 `Physical Computing, Embedded AI, Robotics...`
* 📁 `Processamento de Linguagem Natural, C...`
* 📁 `Visão Computacional`
